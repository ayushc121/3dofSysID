The "AC_VER2_main" file is just a script to call the other files in a more convenient manner.

"model" and "classical_OEM_resultsforAIAA 1" were files used to generate data

AC_VER2_sysID is the main file that implements the equation-error method and output-error methods for parameter identification

AC_VER2_statespace is a supporting file containing the state space model of a 3 degree of freedom fixed wing aircraft
